Crystal Collector Game

Instructions:

You will receive a random number at the start of the game.

There are four crystals. Clicking on each crystal will add a specific amount of points to your total score.

You win the game by matching your total score to the random number. You lose the game, if you don't match the score.

The number of each crystal is hidden until clicked.

The game will change the number of each crystal each time the game starts.
